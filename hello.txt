hello git gitee!
git status :: 查看当前分支信息
git reflog :: 查看当前分支记录
git log :: 详细记录
git add [file]:: 添加文件到工作区
git init :: 初始化仓库
git commit -m [注释] :: 提交暂存区(工作区)的内容到本地仓库
git branch [name] :: 新建分支
git branch :: 查看分支情况
git checkout [name] :: 切换分支
git reset --hard [版本号] :: 切换版本
git merge [分支名] ::合并分支,如果分支冲突,可以编辑冲突文件内容,再次add 后commit(git commit);
git checkout --ours or --theirs [冲突文件名]:: 如果遇到非文本文件,ours意味着保留当前分支的文件版本,theirs保留合并分支的版本;
git remote -v ::查看当前有没有连接远程库
git remote add [别名] [仓库网址] ::添加仓库网址
git push [别名 or 网址] 分支名 :: 推送当前分支到仓库
git pull [仓库别名] [拉取分支] :: 拉取远程仓库分支文件
这是git远程仓库的操作行为
